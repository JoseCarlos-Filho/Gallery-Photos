para gerar um arquivo compilado de SASS para CSS após salvar as alterações 
no projeto, executar a seguinte linha de comando no terminal:
Observação: o terminal tem que estar com a pasta no caminho do projeto.
ex: D:\Projetos_WEB\Projeto_SASS>
comando: sass --watch sass/styles.sass:css/styles.css

repare que este caminho: css/styles.css está na página html:
<link rel="stylesheet" href="css/styles.css">
